1. Mengapa setiap bahasa pemrograman itu terdapat framework, apa kegunaan framework ?
Jawab: Framework merupakan sebuah tools untuk membantu dan mempermudah programer dalam melakukan programming. Kegunaan framework sendiri untuk memisahkan antara model, view dan controller dalam membangun website/program.

2. Jelaskan mengenai fungsi dan penggunaan if else, while, while do, switch case, for, foreach?
Jawab:
  • if else: untuk pemilihan salah satu kondisi yang diinginkan
  • while: untuk melakukan perulangan sesuai kodisi yang diinginkan
  • while do: untuk melakukan pengecekan suatu kondisi dengan menggunakan metode perulangan
  • switch case: untuk memilih suatu kondisi yang diinginkan
  • for: untuk melakukan perulangan sampai pada nilai yang diinputkan
  • foreach: untuk melakukan perulangan, biasanya dipakai untuk pengecekan nilai dari array
3. Apa itu constructor, method, class, object, variabel, parameter, function, procedure ?
Jawab:
  • constructor: method yang otomatis akan berjalan dengan sendirinya pada saat ada sebuat inisiasi
  • method: cara untuk menjalankan suatu fungsi
  • class: tempat untuk inisiasi sebuah object
  • object: suatu variabel atau objek yang merupakan ciri-ciri dari class
  • variabel: tempat untuk inisiasi atau menyimpan suatu nilai
  • parameter: tempat menampung nilai yang akan digunakan pada sebuah fungsi
  • function: sebuah cara untuk menyelesaikan suatu tugas yang ada di dalam program
  • procedure: sebuah langkah-langkah untuk menyelesaikan tugas yang ada pada program
4. Perbedaan method=”Post” dan “Get” ?
Jawab:
  • Post: method yang biasa digunakan untuk mengirimkan data, seperti username, password nama dll tanpa perlu ditampilkan
  • Get: method yang biasa digunakan untuk menampilkan suatu data berdasarkan apa yang diminta
5. Jelaskan konsep mengenai web service dan fungsi apa yang digunakan untuk memanggil web service client ?
Jawab: Web service adalah suatu interkasi antara pengguna dengan aplikasi yang berbasis internet dan database. Fungsi dari web service adalah untuk mempermudah manusia karena dapat diakses di mana saja asalkan memiliki akses internet.

6. Jelaskan mengenai MVC dan HMVC !
Jawab:
  • MVC: Model View Controller, sebuah kerangka yang terdiri dari model sebagai database, view sebagai tampilan dan controller sebagai pengontrolan. MVC digunakan untuk mempermudah karena ke-3 hal tersebut dapat dikerjakan secara terpisah dan sendiri-sendiri
  • HMVC: Sama seperti MVC, namun HMVC lebih praktis dan tertata karena mengusung nilai hirarki
7. Sebutkan fungsi curl() dalam php!
Jawab: Sebuah URL untuk mengakses atau berintertaksi antara PHP dengan API

8. Bagaimana menampilkan sebuah element array dalam PHP ?
Jawab: untuk menampilkan nilai array yaitu dengan menuliskan "print_r($array)" untuk semua nilai. Jika hanya salah satu nilai, menggunakan "$array[0]", sesuai yang diinginkan

9. Perbedaan HTTP dan HTTPS ?
Jawab:
  • HTTP: URL untuk mengakses suatu halaman website
  • HTTPS: URL untuk mengakses suatu halaman website
10. Jelaskan Apa itu API? sebutkan contohnya?
Jawab: sebuah protokol untuk menghubungkan dan berintertaksi antar beberapa program
